|------|----------------|----------------------|------|-------------------------------|

| On-prem / Self-hosted Agent Platforms | ~28.7% of agent platform revenue (2025) | Growing at ~23% CAGR | — | Regulated industries + privacy-conscious users actively seeking alternatives to cloud agents |

### Key Gaps Identified

- Most "local AI agent" tools are still developer-heavy (LangGraph, CrewAI, AutoGen).
- Very few true **family/caregiver-first** products that run fully offline or via simple WhatsApp/SMS with persistent memory and self-improvement.
- Privacy-first trading intelligence agents have demand but lack simple sovereign harnesses.
- Professional presence for sovereign builders is still weak — most GitHub profiles look like cloud-native demos.
